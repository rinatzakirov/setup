Disable Alt drag:
type xfce4-settings-manager in terminal
and then click 'settings editor'
then to the left click xfwm4 channel then in the right pane change the value of easy_click  from <alt> to nothing

Enable alt column selection in Visual Code:
Menu: Selection -> Switch to Ctrl-Click for Multi-cursor

Disable right-mousedown immediate context menu to prevent accidental menu selection with a mouse-up:
sudo apt-get install easystroke
Open Easystroke then go to preferences tab, in Behavior section click Gesture Button
and right click on the rectangular area in Select a Mouse or Pen Button window.
In Timeout Profile dropdown list pick up Timeout Off
Drawing on screen. easystroke show >preferences>Method to show gestures>None
